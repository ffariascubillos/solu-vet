# Solu Vet

Veterinary Practice Management System MVP.

## Commands

Run the API:

```bash
npm run dev:api
```

Run the Expo mobile app:

```bash
cd apps/mobile
npx expo start
```

Check API TypeScript:

```bash
npm exec -w apps/api -- tsc --noEmit
```

Check mobile TypeScript:

```bash
npm exec -w apps/mobile -- tsc --noEmit
```

Run mobile lint:

```bash
npm run lint -w apps/mobile
```
